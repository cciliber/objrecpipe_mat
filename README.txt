Object Recognition Pipeline - MATLAB code -

+Feature: package containing the classes for the feature extraction stage
 - GenericFeature, template for the derived following classes
   - MyDataset, functions for processing the collected images and organizing a dataset
   - MySIFT, SIFT
   - MyPCA, Principal Component Analysis
   - MyFV, Fisher Vectors
   - MySC, Sparse Coding 
   - MyCaffe, wrapper for the Caffe package for CNNs
   - MyOverFeat, wrapper for the OverFeat code
   - MyHMAX, HMAX model

   - MyBOW, Bag Of Words
